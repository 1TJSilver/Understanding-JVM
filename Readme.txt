1. В stack'e создастся примитив int со значением 1

2. Сначала Bootstrap загрузит класс Object в Metaspace, затем в куче создастся объект "o" этого класса,
   и потом в stack'e создастся ссылка на этот объект

3. Сначала Bootstrap загрузит класс Integer в Metaspace, затем в куче создатся объект "ii" этого класса
   со значением 2

4. В stack'e выделится область под медот printAll()

5. В куче создатся объект "uselessVar" типа Integer со значением 700

6. Сначала загрузится класс System в Metaspace, выделится место в stack'e под метод println(),
   затем выделится место в stack'e под метод toString() от объекта "o", по завершению которого,
   выделенное место для него удаляется.

7. Выделенное место под метод printAll() удаляется, а вместе с ним ссылка на объект "uselessVar" и
   место, выделенное под println().
   При следующей очистке мусора, удалится объект Integer, со значением 700.
   Затем в stack'e выделится место под метод println.
